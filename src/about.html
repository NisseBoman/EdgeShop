About EdgeShop

Welcome to EdgeShop - a demo project showcasing the power of Fastly's Compute@Edge platform!

This demo site was created by Nils Boman, a Sales Engineer at Fastly. It serves as a playground to demonstrate the capabilities of edge computing and how it can be used to build fast, responsive e-commerce applications.

As a Sales Engineer at Fastly, Nils is passionate about showing how edge computing can transform the way we build and deliver web applications. This project is just for fun and educational purposes, showcasing various features like:

  • Edge-based routing and processing
  • Dynamic content generation at the edge
  • Image optimization and delivery
  • Fast, responsive user experiences

Note: This is a demo site and not a real e-commerce platform. All products and prices are fictional.
