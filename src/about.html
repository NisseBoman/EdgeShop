Our Story

EdgeShop is a demonstration e-commerce platform built on Fastly's Compute@Edge, showcasing the power of edge computing in modern web applications. Our platform demonstrates how to build fast, secure, and scalable shopping experiences using cutting-edge technology.

Technology Stack

Built with:

  • Fastly Compute@Edge for edge computing
  • Bootstrap for responsive design
  • Modern JavaScript for dynamic functionality
  • KV Store for data management

Features

The code includes:

  • Dynamic product catalog
  • Shopping cart functionality
  • Real-time price calculations
  • Responsive design for all devices
  • Fast image delivery with proper caching
